# 실시간 독서 교정 추가 기능 및 수정 - 05.07

1. 주요 기능 및 개선 내용

    - 의미 단위(Chunking) 하이라이트

    기능:  문장 하이라이트 -> 현재 응시하는 문장의 어절만을 선명하게 강조함

    - 의미 해석 툴팁

    기능: 사용자가 모르는 단어를 보게 되면 자연스레 해당 단어를 응시하게 됨 -> 즉, 사용자가 특정 단어에 1.5초 이상 시선을 고정하면, 해당 단어의 뜻을 알려줌 

    *단점*
        해당 단어의 뜻을 알기 위해선 미리 문장 속 단어들을 추리고 뜻을 작성해둬야 함 (reading.html 파일의 72줄 참고)

    - 문단 요약 위젯

    기능: 사용자가 한 문단의 독서를 마치고 다음 문단으로 넘어가는 시점을 감지, 방금 읽은 문단의 핵심 요약문을
